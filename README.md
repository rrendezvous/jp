## ğŸš€ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (for managing dependencies)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/rrendezvous/jp.git
    cd jp
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

### Development

To watch for CSS changes and compile Tailwind in real-time:

```bash
npm run watch:css
```

To build the CSS for production:

```bash
npm run build:css
```

Open `index.html` in your browser to view the application.
